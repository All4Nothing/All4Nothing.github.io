## CNN-LSTM

CNN과 LSTM을 연결해, 이미지나 동영상 데이터를 통해 텍스트를 추출하는 하이브리드 모델을 만들 수 있다. 이러한 모델은 image captioning에 사용할 수 있다.

CNN을 incoder로 LSTM을 decoder로 구성하여 다음과 같이 학습에 사용할 수 있다.

```python
class CNN(nn.Module):
	...

class LSTM(nn.Module):
	...

# train
for epoch in range(num_epochs):
		for i, img, cap in enumerate(train_dataloader):
				img = img.to(device)
				cap = cap.to(device)
				...
				x = incoder(img)
				outputs = decoder(x, cap)
				loss = criterion(outputs, img, cap)
				incoder.zero_grad()
				decoder.zero_grad()
				loss.backward()
				optimizer.step()
				...
```
