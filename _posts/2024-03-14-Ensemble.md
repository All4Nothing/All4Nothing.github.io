### Ensemble

Ensemble 기법은 여러 개별 모델을 결합하여 더 나은 예측 성능을 얻는 머신러닝 기법이다. 이러한 기법은 데이터를 다양한 방식으로 학습하고 모델 간의 다양성을 확보하여 일반화 성능을 향상시킨다. 그 중에서도 Bagging, Boosting, Stacking은 가장 널리 사용되는 앙상블 기법들이다.

1. **Bagging (Bootstrap Aggregating):**
    - Bagging은 Bootstrap Aggregating의 줄임말로, 데이터 샘플을 여러 번 무작위로 추출하여 각각의 샘플에 대해 독립적인 모델을 학습시키는 기법이다.
    - 각 모델이 독립적으로 학습되기 때문에 병렬 처리가 가능하며, 따라서 학습 속도가 빠르다.
    - 예측 시에는 각 모델이 생성한 예측을 평균 또는 투표 등의 방법으로 결합하여 최종 예측을 만든다.
    - 대표적인 Bagging 기법으로는 Random Forest가 있다.
2. **Boosting:**
    - Boosting은 약한 학습기(weak learner)를 연속적으로 학습시켜 강력한 학습기(strong learner)를 만드는 기법이다.
    - 이전 모델이 잘못 예측한 샘플에 가중치를 부여하여 다음 모델이 더 집중해서 학습할 수 있도록 한다.
    - Boosting은 이전 모델의 오차를 보완하는 방식으로 순차적으로 학습하므로, 학습 시에는 순차적인 진행이 필요하며, 병렬 처리가 어렵다.
    - 대표적인 Boosting 알고리즘으로는 AdaBoost, Gradient Boosting Machine(GBM), XGBoost, LightGBM, CatBoost 등이 있다.
    
    !https://miro.medium.com/v2/resize:fit:1153/1*XzaoQxMf4uLD5DIHz8JwbA.png
    
3. **Stacking:**
- Stacking은 여러 모델의 예측 결과를 다시 학습 데이터로 활용하여 최종 모델을 학습하는 기법이다.
- Stacking은 다단계의 접근 방식을 가지며, 여러 모델로부터 얻은 예측값을 새로운 학습 데이터로 사용하여 다른 모델(메타 모델)을 학습시킨다.
- 다양한 종류의 모델을 조합하여 사용할 수 있으며, 각 모델의 예측 결과를 학습 데이터로 활용하기 때문에 다양한 관점에서 데이터를 학습하게 된다.
- Stacking은 모델 결합에 대한 복잡성이 있고, 적절한 모델의 선택 및 조합이 중요하다. 일반적으로는 교차 검증을 통해 모델 조합을 결정한다.

!https://miro.medium.com/v2/resize:fit:809/0*e-na5r7mF8lVAfPK.png

- CV 기반 Stacking은 Stacking 기법 중에서도 교차 검증(Cross-validation)을 활용하여 모델을 학습하고 예측하는 방법이다. 일반적으로 Stacking은 여러 개의 기본 모델(첫 번째 수준 모델)을 사용하여 예측을 생성하고, 그 예측을 다시 메타 모델(두 번째 수준 모델)에 입력하여 최종 예측을 생성한다. 그러나 CV 기반 Stacking에서는 모델의 학습 및 예측 과정에 교차 검증을 적용하여 모델의 일반화 성능을 더욱 신뢰할 수 있게 만든다
    
    CV 기반 Stacking의 주요 단계는 다음과 같습니다:
    
    1. **첫 번째 수준 모델 학습**: 각각의 기본 모델을 여러 번의 교차 검증을 통해 학습한다. 예를 들어, k-fold 교차 검증을 사용할 수 있다. 각 폴드에서 학습된 모델은 해당 폴드를 제외한 나머지 데이터로 검증되고, 예측을 생성한다.
    2. **첫 번째 수준 모델 예측**: 학습된 각 모델은 전체 훈련 데이터와 테스트 데이터에 대한 예측을 생성한다. 훈련 데이터에 대한 예측은 다음 단계의 메타 모델 학습에 사용된다.
    3. **메타 모델 학습**: 이전 단계에서 생성된 예측 결과를 학습 데이터로 사용하여 메타 모델을 학습한다. 일반적으로는 예측된 값들과 실제 타겟 값 사이의 오차를 최소화하는 방향으로 학습된다.
    4. **최종 예측 생성**: 메타 모델이 학습되면, 각 첫 번째 수준 모델의 예측을 메타 모델에 입력하여 최종 예측을 생성한다.
    
    CV 기반 Stacking은 모델 간의 상호작용을 적절히 고려하면서 일반화 성능을 개선할 수 있다. 또한 교차 검증을 통해 모델의 안정성과 신뢰도를 높일 수 있어, 과적합을 방지하고 예측 성능을 향상시킬 수 있다. 하지만 계산 비용이 증가할 수 있으므로, 큰 데이터셋이나 복잡한 모델 구조에서는 실용적인 측면을 고려하여 사용해야 한다.
    

이러한 앙상블 기법들은 모델의 성능을 향상시키는 강력한 도구로, 데이터셋과 문제에 따라 적절한 기법을 선택하여 활용할 수 있습니다.
